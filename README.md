# claude-ru-gaming

Русские игровые фразы для спиннера Claude Code. Вместо стандартных сообщений при работе Claude показывает культовые фразы из игр — Warcraft, StarCraft, Dark Souls, Factorio, S.T.A.L.K.E.R., Skyrim и других.

## Установка

Скопируйте содержимое `settings.json` в настройки Claude Code:

```bash
# Глобальные настройки
~/.claude/settings.json

# Или настройки проекта
.claude/settings.json
```

Добавьте блок `spinnerVerbs` в ваш файл настроек.

## Пример

Во время работы Claude Code будет показывать фразы вроде:

- *Нужно больше золота*
- *Требуются дополнительные пилоны*
- *Зуг-зуг*
- *Praise the Sun!*
- *Чики-брики, и в дамки*
